OBJECTIF REALISÉ
================================================================================
Intégration complète d'un système de génération automatique de résumés IA pour
le blog Ecobyte, utilisant l'API OpenAI et affichant les résumés dans le 
front-office avec gestion admin pour la génération/régénération.


ARCHITECTURE
================================================================================

Front-office (Blog)
    │
    ├─ blog.php
    │  ├─ Charge ai-summary.css
    │  ├─ Ajoute un conteneur par post
    │  └─ Charge ai-summary.js
    │
    ├─ ai-summary.js
    │  └─ Appelle /api/get_summary.php
    │
    └─ API Backend: get_summary.php
       ├─ Récupère le post en cache
       └─ Ou génère un nouveau via OpenAI
          └─ Sauvegarde en BD


Back-office (Admin)
    │
    └─ admin/summaries.php
       ├─ Liste tous les posts
       ├─ Affiche le statut des résumés
       └─ Permet de générer/régénérer manuellement

INSTALLATION RAPIDE (5 MINUTES)
================================================================================

ETAPE 1 : Configuration API (1 minute)
─────────────────────────────────────
Ouvrez: config_ai.php
Remplacez: define('OPENAI_API_KEY', 'sk-your-actual-key');
Obtenez votre clé: https://platform.openai.com/account/api-keys

ETAPE 2 : Migration Base de Données (1 minute)
───────────────────────────────────────────────
Option A - Via CLI:
  mysql -u root < migration_add_summary.sql

Option B - Via phpMyAdmin:
  1. Allez dans phpMyAdmin
  2. Sélectionnez la base "ecobyte"
  3. Onglet SQL → Copiez le contenu de migration_add_summary.sql
  4. Cliquez Exécuter

ETAPE 3 : Vérification (2 minutes)
──────────────────────────────────
Accédez à: http://localhost/health-check.php
Tous les checks doivent être ✅ VERTS

ETAPE 4 : Test (1 minute)
─────────────────────────
Accédez à: http://localhost/ai_summary_test.php
Testez la génération de résumé

ETAPE 5 : Utilisation
─────────────────────
Front-office: http://localhost/blog.php
Back-office:  http://localhost/admin/summaries.php

DÉPANNAGE
================================================================================

ERREUR: "Unknown column 'summary'"
→ Exécutez la migration SQL (voir ETAPE 2)

ERREUR: "Impossible de générer le résumé"
→ Vérifiez votre clé API OpenAI dans config_ai.php
→ Vérifiez votre limite d'utilisation API

ERREUR: "Résumé ne s'affiche pas"
→ Vérifiez la console du navigateur (F12)
→ Vérifiez que ai-summary.js est chargé
→ Vérifiez que /api/get_summary.php répond

PROBLEME: Aucun résumé généré
→ Exécutez ai_summary_test.php pour diagnostiquer
→ Vérifiez les logs du serveur
→ Vérifiez que cURL est activé
